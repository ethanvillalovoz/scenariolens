# ScenarioLens v0.2.0 Release Notes

ScenarioLens v0.2.0 is the first product-quality public release for the
project: a Waymo-aligned, laptop-friendly framework for mining and analyzing
long-tail autonomy motion scenarios.

## Highlights

- Static Scenario Explorer with filters, trajectory previews, score components,
  and baseline failure summaries.
- Real-data public-safe reports from a local Waymo Motion validation shard.
- Constant-velocity prediction baseline with ADE/FDE, miss rate, and failure
  score.
- Lane-aware prediction baseline comparison for map-backed vehicle/cyclist
  targets, with constant-velocity fallback for unsupported cases.
- `scenariolens baseline-compare` Markdown/JSON reports and a checked-in
  curved-lane fixture where lane-aware FDE improves from 5.831 m to 0.615 m.
- Failure stability workflow for contiguous windows or multiple input shards.
- Shard expansion planner for the next authenticated Waymo downloads.
- Contribution, citation, changelog, and release metadata.

## Try It

```bash
python -m pip install -e .
scenariolens demo
scenariolens report --format markdown --limit 5
scenariolens baseline-compare --format markdown --limit 8
```

## Honest Scope

ScenarioLens is not a self-driving stack, Waymo internal tool, or full-dataset
benchmark. It is a public, reproducible evaluation framework for scenario
triage, baseline failure analysis, and public-safe reporting.
