# ScenarioLens v1.0.0-rc.1 Release Notes

ScenarioLens v1.0.0-rc.1 is the release candidate for the first complete
local-first autonomy evaluation product. It turns the repository's specialist
diagnostics into one reproducible workflow with a generated Explorer, full
local-corpus evidence, a separate frozen-policy holdout, and executable release
gates.

## Highlights

- `scenariolens run` ingests ScenarioLens JSON or local Waymo Motion inputs and
  emits one versioned bundle containing baseline comparison, lane selection,
  linked-lane continuation, ranked trajectory assets, reports, and a static
  Explorer.
- Two complete runs over 1,193 scenarios from four local Waymo Motion
  validation shards produced the same analysis digest, generated 50 ranked
  Explorer cases, and passed all 7 gates. The slower run completed in 601.447
  seconds with 3.642 GB peak memory, within the 15-minute and 8 GB M5 MacBook
  Air budgets.
- A separately frozen 993-scenario holdout evaluated 78 selector decisions and
  passed all 8 integrity gates. The candidate remains disabled because the
  larger cohort exposed 12 false promotions.
- `scenariolens release-check` builds the wheel twice, requires matching
  SHA-256 hashes, installs into a clean virtual environment, runs outside the
  checkout, probes explicit bad-input diagnostics, and verifies interruption
  recovery against an uninterrupted nine-stage run.
- The live Scenario Explorer and generated local Explorer share the same static
  frontend and versioned payload contracts, covered by desktop and mobile
  Playwright workflows.
- Exact x-sweep path-distance indexing removes the quadratic ranking bottleneck
  while preserving byte-identical dashboard output on the 100-scenario
  equivalence probe; the isolated 1,193-scenario Explorer pass dropped from
  324.440 seconds to 134.859 seconds.

## Try It

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install .
scenariolens demo --open
```

Run the release gate from a source checkout:

```bash
scenariolens release-check \
  --repo-root . \
  --output-dir data/processed/scenariolens_v1_release_check
```

## Honest Scope

ScenarioLens is an autonomy scenario-mining and evaluation framework. It is not
a production autonomous-driving stack, a Waymo benchmark submission, a
closed-loop safety certification, or a claim that the experimental selector is
ready for adoption. Raw Waymo records remain local and ignored.

This release candidate becomes `v1.0.0` only after the exact RC commit passes
the full local and CI validation checklist and the deployed Explorer is
verified.
