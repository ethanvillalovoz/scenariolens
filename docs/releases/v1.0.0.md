# ScenarioLens v1.0.0 Release Notes

ScenarioLens v1.0.0 is the first stable release of the local-first autonomy
evaluation framework. It turns local Waymo Motion or ScenarioLens JSON inputs
into reproducible baseline studies, ranked scenario evidence, reports, and a
self-contained static Explorer.

## Promotion Record

The analytical code in `v1.0.0` is unchanged from `v1.0.0-rc.1`. The exact RC
merge commit passed main-branch and annotated-tag CI before this version-only
promotion. The checked-in 1,193-scenario results therefore retain their
`1.0.0rc1` provenance rather than being relabeled after execution.

The final clean-package gate passed 15/15 checks. Two independent builds
produced the same `scenariolens-1.0.0-py3-none-any.whl` SHA-256 digest:
`b09911f6aab3f310fa505931d52f924c9cfe528a3c4eec453130311397067e2c`.

## Highlights

- `scenariolens run` executes baseline comparison, heading-aware lane
  selection, linked-lane continuation, ranking, trajectory rendering, and
  Explorer generation as one versioned product bundle.
- Two complete runs over 1,193 scenarios from four local Waymo Motion
  validation shards produced the same analytical digest and passed all 7
  reproducibility and laptop-budget gates. The slower run took 601.447 seconds
  and 3.642 GB peak memory on the target M5 MacBook Air.
- A separately frozen 993-scenario holdout evaluated 78 selector decisions and
  passed all 8 integrity gates. The candidate remains disabled because the
  larger cohort exposed 12 false promotions.
- `scenariolens release-check` builds a byte-identical wheel twice, installs it
  into an isolated environment, exercises the installed product outside the
  checkout, checks explicit failure diagnostics, and verifies real
  interruption/resume equivalence across a nine-stage workflow.
- The generated and public Scenario Explorers share one static frontend and
  versioned payload contracts, with desktop and mobile browser coverage.

## Install And Run

Download the wheel attached to this release, then run:

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install scenariolens-1.0.0-py3-none-any.whl
scenariolens demo --open
```

From a source checkout:

```bash
python -m pip install .
scenariolens export-synthetic --output /tmp/scenariolens.json
scenariolens run \
  --input /tmp/scenariolens.json \
  --format scenariolens-json \
  --output runs/quickstart \
  --open
```

## Honest Scope

ScenarioLens is an autonomy scenario-mining and evaluation framework. It is
not a production autonomous-driving stack, a Waymo benchmark submission, a
closed-loop safety certification, or evidence that the experimental selector
is ready for adoption. Raw Waymo records and local per-scenario artifacts
remain outside git.
